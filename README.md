# PHP Lab Work

This repository contains lab work and practice programs for the **Web Technology** subject of the **BIM (Bachelor of Information Management) course**.

The project is mainly built with PHP and includes multiple question-wise files organized by lab folders.

## What this project contains

- `lab1` to `lab8` folders with question-based practical assignments (`qno*.php`, `QNo*.php`, etc.).
- `Practice` folder with additional PHP practice files and sample text files.
- Basic session, file handling, form handling, and small web application exercises spread across different lab folders.

Overall, this repository is a collection of BIM Web Technology practical submissions and practice exercises.

## Types of programs in this project (short)

1. Basic PHP fundamentals: variables, operators, conditions, loops, and simple output-based programs.
2. Form handling programs: taking user input through HTML forms and processing it in PHP.
3. Function and logic-based problems: small question-wise programs (qno files) for calculations and pattern/problem solving.
4. File handling tasks: reading/writing text or CSV files and displaying data.
5. Session/login practice: authentication flow with login, logout, and protected pages.
6. Mini web app pages: simple pages like home, signup/login, cart, checkout, and product view.
7. Lab-wise organized exercises: folders from lab1 to lab8 plus extra practice programs.
